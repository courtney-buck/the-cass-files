# Governance

How the agent network makes decisions — and more importantly, how it doesn't.

This folder contains the frameworks that define what agents can do autonomously, what requires human approval, and where the hard lines are. If you're building your own agent system, this is probably the most underrated piece of the puzzle.

## Contents

- `decision-rights-summary.md` — The DO / DRAFT / NEVER framework that governs all agent actions
